## How It Works

1. User enters GitHub username and clicks "Begin Journey"
2. Orb appears from the portal and begins its journey
3. Page auto-scrolls through 5 cinematic scenes
4. In each scene, the orb progressively reveals follower count:
   - Scene 2: 0% revealed
   - Scene 3: ~40% revealed
   - Scene 4: ~70% revealed
   - Scene 5: 100% revealed with crown animation
5. Final follower count displayed with username and achievement text
